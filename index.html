COMPLEXITY ANALYSIS

Theoretical complexity comparison

Single-node cascade detection is O(V+E) either way, but all-nodes precomputation changes from O(V·(V+E)) to O(V+E) with Tarjan-first condensation.

SINGLE-NODE QUERY — SAME COMPLEXITY CLASS, SMALLER CONSTANT

O(V+E) on raw graph
Reverse BFS from the probed node, cycles handled by the visited set.
—
O(V′+E′) on condensed graph
Same BFS, smaller graph — cycle members already known to share fate.
—

ALL-NODES PRECOMPUTE — COMPLEXITY CLASS CHANGES

O(V·(V+E)) naive
Independent BFS per node — can't safely memoize across a cyclic graph.
—
O(V+E) with Tarjan
Condense once, then a single reverse-topological DP pass over the DAG.
